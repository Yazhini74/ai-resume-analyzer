
# AI Resume Analyzer

An AI-powered web application that analyzes resumes and compares them with job descriptions to evaluate candidate suitability. The system provides feedback on skills, keyword matching, and overall resume quality.

🔗 Live Demo: https://resumind-yb.vercel.app/

## Features

* Upload resume (PDF or text)
* Paste job description
* AI-based resume analysis
* Keyword and skill matching
* Resume scoring and feedback
* Clean and responsive UI

## Tech Stack

* Frontend: React / TypeScript
* State Management: Zustand
* Backend: Node.js
* AI Processing: NLP-based analysis
* Styling: Tailwind CSS

## How It Works

1. User uploads a resume.
2. User enters the job description.
3. The AI processes both texts.
4. The system compares skills, keywords, and experience.
5. A match score and suggestions are generated.
